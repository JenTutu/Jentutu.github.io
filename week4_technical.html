Arrays vs. Hashes

Blog 4: Arrays vs. Hashes: What are they? What are they good for?

What is an Array?

An array is simply a list arrangement of objects with no restrictions on type. For instance an array that mixes Integers and Strings is perfectly valid (i.e. array=[1,'one',2,'two']) within Ruby.

Creating an array is fairly simple. You can build an array by placing a set of values within brackets[], i.e.:

  • [1,2,3]

More often than not, you will want to name the array so that you can access it later.

  • named_array=[1,2,3]
    #name_of_array=[object1,object2]

What is a Hash?

A hash, similar to an array, is simply a collection of objects in no particular order.

Typically, hashes are used to represent items and their associated values. For instance, if I wanted to be able to access a list of books by author, I could do so like this:

books = {}
books[:matz]= "The Ruby Lang"
books[:black] = "The Well-Grounded Rubyist"
#name of hash[:name of key]= "value"

This brings me to another point. How do you create a Hash? There are two ways:

    Example 1: Using Hash Literal
  • ages={"Genevieve Dior" => 24, "Naomi Cisse"=>26}
    #name of hash= {"key1"=> value1, "key2"=> value2}
    Example 2: Using ::new method
  • ages=Hash.new
    ages["Genevieve Dior"]=24
    #name of hash["name of key"]=value

When to Use an Array vs. a Hash

Not surprisingly, deciding when to use an Array over a Hash will be dependent on the situation. While both arrays and hashes are data structures, hashes are an excellent choice for when you want to displate items that are associated with one another. An example can be found in displaying menu items with their associated costs:

    Example 3: Menu Items & Prices restaurant_menu = { "Ramen" => 3, "Dal Makhani" => 4, "Coffee" => 2 }
    restaurant_menu.each do | item, price |
    puts "#{item}: $#{price}"
    end
      => Dal Makhani: $4
      Coffee: $2
      Ramen: $3

Since arrays are useful for storing one-dimensional data, they are useful for looking up data using an index.

    Example 4: An Array of a student group
    studentArray = ["Ajay Bir Singh", "Jennifer Oseitutu", "Andrew Principe", "Joseph Rosztoczy"]

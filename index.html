Projects

As part of an 8-day project at Dev Bootcamp, I worked on third party integration for a real-time web app service called DivvyUpTours. DivvyUpTours is a Rails and JavaScript web app that lets users combine the adventure of a Divvy bike ride with sightseeing in Chicago. Using an API of the Divvy API, I was able to integrate real-time information on the nearest Divvy bike stations, along with the number of available bikes and bike docks.

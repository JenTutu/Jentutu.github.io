(Week 8) What is jQuery?

Good question! jQuery is a Javascript framework; in short, jQuery makes it easier to write Javascript. Thus, you can use jQuery to make your websites more interactive and even include animations!

Many big name companies use jQuery for their own sites, including WordPress, IBM, and Twitter. Since jQuery allows you to add functionality to your website, you can add features such as the ability to load new information without having to refresh the page.
